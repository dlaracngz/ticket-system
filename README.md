# ticket-system
Kullanıcıların talep oluşturup takip edebildiği ve yorum ekleyebildiği full-stack ticket yönetim sistemi.
Projede backend kodunu çalıştırabilmeniz için ana dizine cd backend ve ardından npm start komutunu yazmanız yeterli.
Projede frontend kodunu çalıştırabilmeniz için başka bir terminal açıp cd frontend ardından npm run dev yazmanız yeterli.
